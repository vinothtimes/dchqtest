-------
! ABOUT
-------

JsTetris is a tetris clone game. It is written in javascript, css, html. It only requires a browser to run.
Tested on: IE, Mozilla, Opera.

You can subscribe to new releases here:
http://freshmeat.net/projects/jstetris

This script can be used freely as long as all copyright messages are intact.
